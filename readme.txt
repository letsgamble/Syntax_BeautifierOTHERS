##################### Overview #####################
SyntaxBeautifierOTHERS is a tool created
for the automated transition of Syntax
between Keyword and Machine Learning class

##################### Features #####################
-If AttributesContain have less than 3 attributes
and don't have distancer, it will become SimpleKeyword
-If AttributesContain have less than 3
attributes and have distancer, it will become
SpecialKeyword with distancer broaden to :any:
-Moreover, all distancers will get broaden to :any:

###################### Usage ######################
Simply paste the syntax into Initial Syntax textarea
and hit Submit button.
Output subpage with two buttons should appear:
-Copy text - copy output syntax into the clipboard
-Reset - returns to the homepage.
